Accès pour les Grands Voisins

Chaque personne peut bénéficier d'un compte Les Grands Voisins. Ce compte permet une gestion de services annexes: entrée dans l'annuaire, courriel, ...

Votre compte

Pour vous connecter ou pour créer votre compte, rendez-vous sur key.lesgrandsvoisins.com.

  • Les applications accessibles avec votre compte sont visibles dans la rubrique apps
  • La création d'un compte est accessible par le lien "Nouvel utilisateur ? Enregistrement" vers le bas de la fenêtre avec la possibilité en bas d'utiliser un compte Google ou un compte hotmail existant. (Attention, merci de bien penser à cliquer à l'intérieur du courriel de vérification pour bien enregistrer votre compte dans Les Grands Voisins.)

Votre entrée dans l'annuaire

Pour pouvoir modifier votre fiche dans l'annuaire, il convient de s'identifier dans le site de contenu de l'annuaire lesgrandsvoisins.com/cms-admin. Votre compte sera techniquement créé à la première connexion si ce n'est pas encore le cas. Suite à cela, l'administration liera manuellement votre compte avec votre enregistrement dans l'annuaire.
